Overview

Designed and developed an e-commerce application using the JSONProducts API, focusing on creating a user-friendly interface and implementing essential e-commerce features.

Designed and developed an e-commerce application focused on fetching API details, displaying products, and implementing filtering based on price, rating, and categories. The application allows users to navigate to each product to see more images and view a product image gallery.

Key Contributions:

Fetching API Details : Fetched product details from the API to display them on the application.

Home Page Design: Designed the home page with login and logout functionality to avoid unnecessary navigation. Products are displayed upon login, and users are navigated to a separate page upon logout.

Product Navigation: Used React Router for seamless navigation. Each product's information is displayed with the ability to change images by clicking on thumbnail images.

Product Information Display: Displayed comprehensive information for each product, including large images that change based on user interaction with smaller thumbnail images.

Filtering Options: Added filtering options to sort products based on various criteria such as price (low to high, high to low) and categories (e.g., smartphones, groceries). Provided brand details and categorized products for easy browsing.

Technologies Utilized:

HTML/CSS: Employed front-end technologies to customize the website's appearance and functionality, ensuring a seamless and visually appealing user experience across all devices.

Bootstrap: Utilized Bootstrap to create a responsive design, ensuring that the application is accessible and visually appealing on a wide range of devices, including desktops, tablets, and smartphones.

React: Implemented various functionalities using React, including fetching and displaying product details from APIs. Designed a home page with login and logout functionality to manage user navigation. Leveraged React Router for seamless product navigation, enabling users to view detailed information and image galleries for each product. Added filtering options to sort products based on price, rating, and categories, providing a dynamic and interactive user experience.

Results: Developed an e-commerce application that successfully incorporates key features of a modern shopping platform. The implementation of product details, price and category-based filtering, and seamless navigation provided a comprehensive and user-friendly experience. The application allows users to view detailed product information and image galleries, enhancing their shopping experience. The visually appealing design, intuitive navigation, and optimized performance contributed to an improved user experience, driving increased engagement and usability of the app.
